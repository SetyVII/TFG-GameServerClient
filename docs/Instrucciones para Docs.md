## Intrucciones para la realización de la documentación

- Portada: Título del proyecto, Logotipo del instituto, autores y coordinadores.
- Indice paginado: que funcione los enlace para ir a la página.
- Encabezados
- Píes de página con la numeración de las páginas
- Tipo de letra Time News Roman o Arial de 12
- Interlineado 1,5.
- Las imagenes deben tener un título.
- Apartado Agradecimientos.
- Bibliografía con enlaces tipo APA.
